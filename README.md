# Social-Media-Addiction-Analysis-SQL
Project Overview

This project focuses on analyzing social media usage patterns and addiction levels among students across different countries and demographics. The objective is to understand how excessive social media usage impacts sleep patterns, mental health, relationship status, and daily behavior using SQL-based data analysis.
